# **App Name**: CoinCatcher

## Core Features:

- Login and Registration: Secure login and registration with email verification and Google sign-in.
- Reward Claim: Users can claim rewards every three hours using smart links.
- Coin System: Manage user balances using a coin based virtual currency.
- Ad Integration: Display banner ads and Popunder ads provided by Adstera for monetization.
- Referral program: Reward existing and new user when they use the referral link to install and start using the app.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of value and modernity.
- Background color: Light gray (#F0F0F0), a desaturated purple, for a clean and modern feel.
- Accent color: Teal (#009688), an analogous color, to highlight interactive elements and calls to action, giving a contrasting pop.
- Font: 'Inter' (sans-serif) for both headlines and body text to ensure readability and a clean user interface.
- Use simple, outlined icons for clear visual cues.
- Prioritize a clear, tab-based layout for easy navigation between reward claiming, coin balance, and settings.
- Use subtle animations, such as coin animations upon claiming rewards, to provide engaging feedback.
